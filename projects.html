Scrambled eggs is a website that teaches you how to make delicious scrambled eggs. It tells you about the ingredients you need and instructions to make scrambled eggs. Click on the link above to go to scrambled eggs. I hope you enjoy!

Snake is a game using html and javascript. It's basically a game that there is a snake and an apple. You can use the arrow keys to control the snake. Every time you eat an apple, the snake will grow longer. If, at some point, the snake's head touches it's tail, the game starts over.

Alien invasion is a python game using python. You basically need to control the ship by using the left and right arrows. Press the space bar to shoot, and shoot down the alien invasion. Every time you shoot down a round, everything gets faster. The ships on the top left corner are how many ships there.

Alien volcano is a simulator that uses python. It basically simulates a moon volcano erupting. It shows the kind of gas that there is. The eruption basically starts small and gets big like normal volcanos. When the eruption goes everywhere, it stops getting bigger.
