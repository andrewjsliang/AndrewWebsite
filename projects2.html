Bounce is a game using python. You basically use the arrow keys to move the paddle left and right. You have to use the paddle to keep the ball in the air. If the ball touches the ground, the game is over. This was from a book tutorial.

Hangman is a game using html and javascript. Basically, you guess a word by typing letters in a random word and you keep type words until you guessed it. This is from a coding class homework.

Mars orbiter is a simulater that uses python. It basiacally simulates a satellite orbiting mars. It shows that the satellite has an error and you can use the arrow keys to correct the orbit. If the satellite gets to far or enters the atmosphere, the orbit fails. This was from a coding class homework.

Treasure hunt is a game that uses HTML and javascript. This game is like a treasure hunt. You click on the map to guess where the treasure is. Treasure hunt uses the hot and cold clue. Basically, when it writes cold, really cold, or freezing, it tells you that you are far away from the treasure. When it says warm, hot, really hot, or boiling hot, that means that you are close to the treasure. Once you clicked the treasure, it will say that you found the treasure in how many clicks it took for you to find the treasure. Click on next round to change the treasure spot. This is also from coding class homework.
