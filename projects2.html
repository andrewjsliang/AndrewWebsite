Bounce is a game using python. You basically use the arrow keys to move the paddle left and right. You have to use the paddle to keep the ball in the air. If the ball touches the ground, the game is over.

Hangman is a game using html and javascript. Basically, you guess a word by typing letters in a random word and you keep type words until you guessed it.

Mars orbiter is a simulater that uses python. It basiacally simulates a satellite orbiting mars. It shows that the satellite has an error and you can use the arrow keys to correct the orbit. If the satellite gets to far or enters the atmosphere, the orbit fails.
